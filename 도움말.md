# 이미지 편집기 도움말

브라우저에서 동작하는 간단한 이미지 편집기입니다. 배경 비트맵 위에 선·도형·텍스트를 겹쳐 그리며, 저장 시 한 장의 이미지로 합쳐 보냅니다.

## 화면 구성

- **상단 도구 모음**: 파일, 도구, 색·선 두께, 실행 취소/다시 실행, 확대·축소, 복사·붙여넣기, 색상 팔레트
- **중앙 캔버스**: 편집 영역
- **하단 상태 표시줄**: 파일 이름, 이미지 크기, 선택 영역 좌표·크기, 배율

## 파일

| 동작 | 방법 |
|------|------|
| 열기 | **열기** 버튼, 또는 **Ctrl+O** / **⌘+O** |
| 이미지 끌어다 놓기 | 캔버스 위로 이미지 파일을 드래그 앤 드롭 |
| 클립보드에서 새 이미지 | **클립보드에서 새 이미지로 열기** (현재 문서 대신 새 캔버스로 붙여넣기) |
| 저장 (PNG 기본) | **저장** 버튼, 또는 **Ctrl+S** / **⌘+S** |
| 다른 이름·포맷으로 저장 | **다른 이름으로 저장** 또는 **Ctrl+Shift+S** / **⌘+Shift+S** — PNG, JPEG, WebP, 품질(JPEG/WebP) |
| 이미지 크기 조절 | 픽셀 크기를 바꾸며 **리샘플**(내용을 새 크기에 맞게 그리기) |
| 캔버스 크기 조절 | 캔버스 가로·세로를 바꾸며 이미지를 늘리거나 줄임. 도형 좌표는 비율에 맞게 함께 조정됨 |

## 보기·이동

- **마우스 휠**: 포인터 위치를 기준으로 확대·축소(약 1%~800%)
- **가운데 버튼 드래그** 또는 **Alt + 왼쪽 버튼 드래그**: 캔버스 **팬**(이동)
- **맞춤** 버튼: 배율 100%에 가깝게 맞추고 위치를 초기화

## 도구별 사용법

### 선택 박스

드래그로 **직사각형 선택**을 만듭니다. 선택 영역은 복사·잘라내기·붙여넣기에 사용됩니다.

### 선·사각형·원

한 번 **누른 채 드래그**하여 그립니다. 마우스를 놓으면 도형이 확정됩니다.

### 폴리라인

**클릭**마다 꼭짓점이 추가됩니다.

- **Enter**: 완료(점이 2개 이상일 때)
- **우클릭**: 완료(점이 2개 이상일 때)
- **Escape**: 그리기 취소

### 자유그리기

**첫 클릭**으로 시작한 뒤 **마우스를 움직이며** 선을 그립니다. **다시 클릭**하면 확정됩니다(점이 충분할 때).

- **Enter**: 완료
- **Escape**: 취소
- **우클릭**(점이 2개 이상): 완료

### 페인트통(영역 채우기)

클릭한 위치와 같은 색으로 연결된 영역을 현재 **전경색**으로 채웁니다. (도형 외곽선은 채우기 판단에 함께 반영되며, 채우기 후에는 해당 도형이 비트맵에 합쳐진 상태로 정리됩니다.)

- **톨러런스**(0~100): 색이 얼마나 달라도 같은 영역으로 볼지 조절합니다.
- **알파 무시**: 켜면 채우기 판별 시 RGB만 비교해 반투명·안티앨리어싱 경계에 유리할 수 있습니다.
- 외부에서 불러온 이미지가 **교차 출처(cross-origin)** 로 픽셀을 읽을 수 없으면 채우기가 동작하지 않을 수 있습니다.

### 텍스트

캔버스를 **클릭**해 위치를 정한 뒤, 하단 패널에 입력합니다.

- **확인** 또는 **Ctrl+Enter** / **⌘+Enter**: 캔버스에 반영
- **취소** 또는 **Escape**: 입력 취소
- 캔버스를 다시 클릭하면 **배치 위치만** 바꿀 수 있습니다.

## 색·선

- **색상**: 기본 색 선택기, **고급 색상 선택**(그라데이션 영역), 오른쪽 **고정 팔레트** 클릭
- **선 두께**: 1~50
- **텍스트 크기**: 텍스트 도구 선택 시 8~256(px)

## 복사·붙여넣기

- **복사** / **Ctrl+C** / **⌘+C**: 선택 영역이 있을 때, 클립보드에 이미지(PNG)로 복사
- **잘라내기** / **Ctrl+X** / **⌘+X**: 복사 후 선택 영역을 배경 이미지에서 지움
- **붙여넣기** / **Ctrl+V** / **⌘+V**:
  - **선택이 있으면**: 그 안에 이미지를 맞춰 합성
  - **선택이 없으면**: 현재 이미지 중앙에 배치
  - **클립보드에서 새 이미지로 열기**: 새 문서로 붙여넣기

입력란(텍스트 영역 등)에 포커스가 있을 때는 전역 붙여넣기가 가로채지지 않을 수 있습니다.

## 실행 취소·다시 실행

- **Undo**(되돌리기) 버튼: 마지막 **도형 추가** 또는 **이미지 변경**(페인트통, 붙여넣기 등)을 한 단계 되돌립니다.
- **Redo**(다시 실행) 버튼, 또는 **Ctrl+Shift+Z** / **Ctrl+Y** (**⌘** 조합 포함): Undo로 빼낸 **이미지 상태**를 한 단계 복구합니다.
- 이미지 **붙여넣기 직후**에는 **Ctrl+Z** / **⌘+Z**가 **마지막 붙여넣기**만 우선적으로 되돌리도록 동작할 수 있습니다. 그 외에는 주로 **Undo 버튼**을 사용하세요.
- **모든 도형 삭제**: 도형만 비우고, 도형 관련 실행 취소 기록은 정리됩니다.

## 저장 시 참고

- **JPEG** 저장 시 투명 영역은 흰색 배경 위에 합쳐집니다.
- 최종 파일에는 배경 이미지와 도형·텍스트가 한 장으로 합쳐져 저장됩니다.

## 실행 환경

- 최신 **Chrome**·**Edge**·**Firefox** 등을 권장합니다.
- 클립보드 읽기/쓰기는 **HTTPS** 또는 **localhost**에서 더 안정적으로 동작하는 경우가 많습니다.
